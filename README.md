# AI Contract Editor

An AI-powered web app that reviews, edits, and highlights contract changes using FastAPI and Groq LLMs.

## Features
- Upload DOCX contracts
- AI-generated clause edits
- Highlighted changes
- Editable contract preview
- Download revised DOCX

## Tech Stack
- FastAPI
- Jinja2
- Groq LLM
- Python-docx
- HTML/CSS/JS
